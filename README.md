# ✨ قـافـيـةAI (QafiyaAI)

منصة ذكية لتوليد القصائد العربية باستخدام تقنيات الذكاء الاصطناعي، تجمع بين تجربة مستخدم عصرية وتصميم عربي أنيق.

---

## 🚀 الفكرة

تهدف المنصة إلى تمكين المستخدم من:

* توليد قصائد عربية بسهولة
* تجربة فورية بدون تسجيل دخول
* حفظ القصائد والمحادثات بعد تسجيل الدخول
* بناء مكتبة شخصية من الإبداعات

---

## 🧩 المميزات الحالية

* 🧠 واجهة شات تفاعلية (مشابهة لتجربة ChatGPT)
* 🎨 تصميم حديث باستخدام خطوط عربية (Cairo + Amiri)
* 📂 نظام تبويبات (مولد - قصائدي - المفضلة - البروفايل)
* 🧾 صفحة توليد عامة بدون تسجيل دخول
* 💬 نظام محادثات مع Sidebar
* ❤️ نظام مفضلة
* 📥 تحميل القصائد كملف نصي
* 🔐 نظام تسجيل دخول باستخدام Firebase Authentication

---

## 🛠️ التقنيات المستخدمة

* HTML5 / CSS3 / JavaScript (Vanilla)
* Firebase Authentication
* LocalStorage (مؤقت لتخزين المحادثات)
* Google Fonts

## ⚙️ كيفية التشغيل

1. تثبيت XAMPP وتشغيل Apache
2. وضع المشروع داخل:

```
C:\xampp\htdocs\
```

3. فتح المتصفح:

```
http://localhost/project
```
